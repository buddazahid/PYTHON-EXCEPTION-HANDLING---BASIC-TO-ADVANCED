# Python Exception Handling - Basic to Advanced

This repository contains a Python tutorial script that demonstrates exception handling concepts from basic to advanced levels.

## File

- `PYTHON EXCEPTION HANDLING - BASIC TO ADVANCED.PY`
  - Shows how to use `try`, `except`, `else`, `finally`, and `raise`
  - Covers common exceptions such as `ZeroDivisionError`, `ValueError`, `IndexError`, `KeyError`, `TypeError`, and `FileNotFoundError`
  - Demonstrates multiple `except` blocks, exception variables, nested `try` blocks, assertion handling, and custom exceptions
  - Includes real-world-style examples like percentage calculation, ATM withdrawal checks, and a login system

## How to run

1. Make sure Python is installed on your system.
2. Open a terminal in this repository folder.
3. Run:

```bash
python "PYTHON EXCEPTION HANDLING - BASIC TO ADVANCED.PY"
```

## Purpose

This script is intended for Python learners who want to understand exception handling and error management through practical code examples.
